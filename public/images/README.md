# Hướng dẫn copy ảnh trường đại học

Vui lòng copy các file ảnh trường đại học vào thư mục này với tên như sau:

## Danh sách ảnh cần có:

1. **dh_fptu.jpg** - Ảnh Trường Đại học FPT Hà Nội
2. **dh_hvtc.jpg** - Ảnh Học viện Tài chính  
3. **dh_vnu.jpg** - Ảnh Đại học Quốc gia Hà Nội
4. **dh_thanglong.jpg** - Ảnh Trường Đại học Thăng Long
5. **dh_hoalac.jpg** - Ảnh Đại học Điện Lực cs4

## Yêu cầu ảnh:
- **Định dạng:** JPG, PNG hoặc WebP
- **Kích thước khuyến nghị:** 400x300px trở lên
- **Tỷ lệ:** 4:3 (landscape)
- **Dung lượng:** < 500KB mỗi ảnh

## Cách copy:
1. Copy các file ảnh từ thư mục `images/dh_*` hiện tại
2. Paste vào thư mục `public/images/` này
3. Đổi tên theo đúng format trên (nếu cần)

Sau khi copy xong, các ảnh sẽ tự động hiển thị trong:
- Homepage section "Khám phá những phòng trọ gần các trường học"
- Trang chi tiết từng trường đại học
